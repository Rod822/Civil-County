# Civil-County

- **Гравець** – користувач на сервері.
- **Профіль** – дані гравця (persistent + session).
- **PersistentData** – збережений баланс (Money).
- **SessionData** – тимчасові дані (job, bonus).
- **Робота (Job)** – роль із командою та зарплатою.
- **Команда (Team)** – Roblox-об'єкт, групує гравців.
- **Зарплата (Paycheck)** – basePay + bonus.
- **Trie** – структура для пошуку за префіксом.
- **DataStore** – Roblox база для збереження.
- **RemoteEvent** – канал сповіщень сервер→клієнт.
