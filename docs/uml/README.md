# UML Діаграми - Civil-County

У цій папці знаходяться UML діаграми архітектури системи Civil-County.

## Діаграми

### 01-use-case.puml
Основні варіанти використання: вхід на сервер, перегляд балансу, призначення на роботу, отримання зарплати, сповіщення та пошук.

### 02-class-diagram.puml
Структура класів: Profile, PersistentData, SessionData, TrieNode та сервіси.

### 03a-activity-player-join.puml
Вхід гравця: завантаження даних, створення профіля, ініціалізація leaderstats.

### 03b-activity-job-assignment.puml
Призначення на роботу: перевірка, оновлення інструментів, відправка сповіщення.

### 03c-activity-paycheck.puml
Розрахунок зарплати: базова сума, бонуси, додання грошей, сповіщення.

### 04a-sequence-player-join.puml
Послідовність входу гравця у систему.

### 04b-sequence-job-assignment.puml
Послідовність призначення на роботу.

### 04c-sequence-paycheck.puml
Послідовність розрахунку та виплати зарплати.

Діаграми створені у PlantUML та можуть бути генеровані у PNG або SVG.
